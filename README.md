# Play-A-Role__A-MBTI-web-Game
A lightweight web game about playing MBTI characters.
一个关于扮演mbti形象的轻量网页小游戏。

游戏中玩家可以选一个mbti人格来进行‘扮演’，通过和其他角色互动和选择，来强化或弱化自己人格特点。
游玩中，你可以选择顺从自己所做行为就是发自内心、也可以为了“更像自己”而精心思考合适回应、甚至可以根据选择和结果来自我判断自己到底是怎样的人格。
但无论如何，只要你自己才最清楚自己是什么样的人，不是吗？

基于H5/css/js制作的小游戏，也算是“3A”大作（AI编程、AI作画、AI作曲），欢迎大家体验和反馈。
